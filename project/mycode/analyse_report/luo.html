摘要
本文通过对饿了么评论数据进行分析研究。采用统计方法，以Python为工具，结合业务知识，将大学生奶茶购买划分为产品搭配、消费者购买产品价格、消费者购买、重复购买动机四大方面进行研究。 最终为商家针对消费者购买行为进行针对性的改进

研究背景及意义

在线外卖普遍化、外卖竞争激烈


研究背景及意义

奶茶价格弹性高、非刚性需求

对于商家来说意味着有更多的调整空间

帮助商家更好的了解消费者


理论
销售额 = 客单价 x 客流量
= （客单品数 x 单品均价）x （新客户 + 回头客）

为了提高销售额，商家应该根据消费者的购买习惯、价格偏好、以及购买动机进行研究。结合奶茶外卖的特征，归纳出相关因素如下：消费者购买的主食（即奶茶、小吃）的件数、消费者购买配料（如奶盖、布丁）的数量、消费者购买的可能动机（价格、好评、销售等）、消费者重复购买的原因（价格、好评、店家、产品等）。

理论
根据以上理论可以归纳为：

产品搭配问题、价格问题、
消费者购买问题、重复购买问题

本文所使用数据采用python编程语言编写爬虫器，对饿了么平台商家、评论、菜单三方面数据进行爬取。获得的数据为json格式，本文通过Python进行数据处理，去重，汇总成表。并通过分组，合并的方式整合出适合本文分析的矩阵表。
